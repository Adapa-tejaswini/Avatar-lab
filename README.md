🎭 Avatar Lab – AI Talking Avatar Generator

A full-stack AI application that generates lifelike talking avatars from images and text. The system converts text into speech and synchronizes lip movements with the generated audio to produce realistic talking avatar videos.

Built using modern AI models and web technologies, the platform demonstrates the integration of speech synthesis, facial animation, and full-stack development.

📌 Overview

Avatar Lab allows users to:

Upload an image of a person

Convert text into speech

Synchronize lip movements with the generated speech

Generate a realistic talking avatar video

The system integrates text-to-speech models with lip synchronization AI to produce high-quality animated avatars.
